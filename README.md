#  Swiggy Sales Analysis | End-to-End Data Analytics Project
This project delivers a complete data analytics solution using SQL, Excel, and Power BI to transform raw food delivery data into meaningful business insights.
It covers data cleaning, modeling, KPI creation, and interactive dashboard development for data-driven decision-making.
________________________________________
## Project Overview
This project analyzes food delivery data across multiple dimensions such as location, restaurant performance, customer spending, and ratings.
The goal is to build a scalable analytics system and create interactive dashboards to track key performance indicators.
________________________________________
## Tech Stack
•	🗄️ SQL – Data Cleaning, Transformation, Star Schema Modeling

•	📊 Excel – Data Validation & Preprocessing

•	📈 Power BI – Data Modeling, DAX, Dashboard Visualization
